# 学生现场报名叫号系统（Web）

基于 Node.js + 原生前端实现的现场报名叫号系统，覆盖：

- 学生扫码取号（免登录）
- 等候区实时叫号展示
- 登记老师窗口叫号（下一号/重呼/过号/完成/重新排队）
- 管理员后台配置（系统名称、流水码前缀、报名类型、窗口数量等）

## 主要能力（一期）

- **扫码取号**：学生填写姓名/手机号/报名类型后，生成唯一流水码与简码（如 `BM20260408-001`、`A001`）
- **队列管理**：自动进入等待队列，支持状态流转：待叫号、已叫号、办理中、已完成、过号、取消、异常
- **老师叫号台**：按顺序叫号，支持重呼与异常处理
- **大屏信息**：显示当前各窗口叫号与最近叫号记录
- **管理配置**：可配置报名类型、号段前缀、叫号参数等

## 启动

```bash
npm install
npm start
```

打开：`http://localhost:3000`

## 默认账号

- 管理员：`admin / admin123`
- 登记老师：`teacher / teacher123`

学生扫码取号无需登录。
