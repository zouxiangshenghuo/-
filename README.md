# 新生报名叫号系统（Web）

一个可在校内电脑浏览器直接使用的新生报名叫号系统，包含：

- **报告厅界面**：实时显示当前叫号流水码、登记处编号、老师姓名、系统名称，并展示最近叫号记录。
- **登记处界面**：老师选择登记处编号后点击“叫号”，系统按流水码从小到大分配叫号。
- **管理员后台**：
  - 流水码区间可配置（1-2000）
  - 语音播报次数可配置（1-10）
  - 最近叫号记录显示条数可配置（1-100）
  - 登记处数量可配置（1-100）
  - 维护老师名单并通过下拉选择分配给登记处（每个登记处最多 2 位老师）
  - 过号可手动添加号码，重新进入待叫号队列（自动按号码升序）
  - 系统名称、字体、字号、颜色可配置
- **账号管理**：管理员 / 普通账号。

## 启动

```bash
npm install
npm start
```

访问 `http://localhost:3000`。

默认账号：

- 管理员：admin / admin123
- 普通账号：user / user123
